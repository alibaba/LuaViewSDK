全局函数

Label ()
通过这个全局静态函数 构建一个新的Label对象

返回值:

    Label label

实例方法

text (text)
设置Label文本信息

参数:

  • text string 文本

例子:

     local label = Label();
 label.text("测试");
text ()
获取Label文本

返回值:

    string text 文本
textColor (color, alpha)
设置Label文本颜色

参数:

  • color int 色值
  • alpha number 透明度
textColor ()
获取Label文本颜色

返回值:

  1. int color 色值
  2. number alpha 透明度
font (fontName, fontSize)
设置Label字体

参数:

  • fontName string 可选参数
  • fontSize number
font ()
获取Label字体

返回值:

  1. string fontName
  2. number fontSize
fontSize (fontSize)
设置Label字体大小

参数:

  • fontSize number
fontSize ()
获取Label字体大小

返回值:

    number fontSize
textSize (fontSize)
设置Label字体大小

参数:

  • fontSize number
textSize ()
获取Label字体大小

返回值:

    number fontSize
textAlign (align)
设置Label文本对齐方式

参数:

  • align int 可选参数

例子:

     TextAlignment
 TextAlign.LEFT -- 左对齐,上下居中
 TextAlign.RIGHT -- 右对齐,上下居中
 TextAlign.CENTER -- 左右居中,上下居中
 cell.nameLabel = Label()
 cell.nameLabel.lineCount(2)
 cell.nameLabel.textAlign(TextAlign.LEFT)
 cell.nameLabel.backgroundColor(self.colors.white, 0)
textAlign ()
获取Label文本对齐方式

返回值:

    int align对齐方式
lineCount (num)
设置Label最多行数

参数:

  • num int 行数

例子:

     TextAlignment
 TextAlign.LEFT -- 左对齐,上下居中
 TextAlign.RIGHT -- 右对齐,上下居中
 TextAlign.CENTER -- 左右居中,上下居中
 cell.nameLabel = Label()
 cell.nameLabel.lineCount(2)
 cell.nameLabel.textAlign(TextAlign.LEFT)
 cell.nameLabel.backgroundColor(self.colors.white, 0)
lineCount ()
获取Label最多行数

返回值:

    int num 行数
adjustFontSize (yes)
让Label字体大小适应宽度, 设置的字体为最大字体, 如果文字超出会缩小字体.

参数:

  • yes bool 是否自适应字体大小(Label的边框固定,如果字数太多字体会缩小保证显示完整)

例子:

     cell.infoRight = Label();
 cell.infoRight.frame( cellW-labelW-dx, y0, labelW, barH);
 cell.infoRight.adjustFontSize(true);
 cell.infoBackground.backgroundColor(0xfff1d1);
 cell.infoRight.textAlign(TextAlign.RIGHT);
