全局函数

TextField ()
通过这个方法创建一个TextField对象

返回值:

    TextField textField

例子:

    local textField = TextField();

实例方法

text (s)
设置文本内容

参数:

  • s string text
text ()
获取文本内容

返回值:

    string s text
placeholder (s)
设置palceHolder内容

参数:

  • s string text
placeholder ()
获取placeHolder内容

返回值:

    string s text
hint (s)
设置hint内容

参数:

  • s string text
hint ()
获取hint内容

返回值:

    string s text
callback (delegate)
设置delegate

参数:

  • delegate function 代理方法

例子:

     textField.callback(
    {
         BeginEditing = function() --开始编辑
         end,
         EndEditing = function() -- 结束编辑
         end,
         Clear = function() -- 清理
         end,
         Return = function() --返回
         end
    }
 );
