全局函数

Image ()
通过这个方法创建一个Image对象

返回值:

    Image view

例子:

    local view = Image();

实例方法

image (url)
设置ImageView的图片url

参数:

  • url string image url

例子:

     local view = Image();
 view.image("https://gju2.alicdn.com/bao/uploaded/TB25bF4eFXXXXbzXpXXXXXXXXXX_!!0-0-juitemmedia.jpg_560x560Q90.jpg");
startAnimationImages (images, duration, repeatCount)
开始帧动画

参数:

  • images array 帧图片数组
  • duration number 动画时间
  • repeatCount int 重复次数

例子:

     scrW, scrH = System.screenSize();
 window.backgroundColor(0x777777,1);
 w = scrW/3;
 X0 = scrW/10;
 Y0 = scrH/5;
 imageView = Image();
 imageView.frame(X0, Y0, w, w);
 imageView.backgroundColor(0xff0000,1);
 names = {};
 for i=1,16 do
 	names[i] = "img" .. i .. ".tiff"
 end
 imageView.image("img16.tiff");
 imageView.startAnimationImages( names, 3.0, 1000);
stopAnimationImages ()
停止帧动画

例子:

    imageView.stopAnimatingImages();
isAnimationImages ()
是否正在播放帧动画

返回值:

    bool 是否正在播放帧动画
